# CapEvent AI Development Workflow

This workflow is intentionally constrained to the requested scope and was followed for this delivery.

1. **Define Scope**
   - Build only the requested features: `Moment Capture` and `Daily Summary`.
   - Avoid adding navigation flows or unrelated screens.
2. **Set Up App Skeleton**
   - Configure Expo + React Native TypeScript project files.
3. **Implement Local Storage Layer**
   - Add a local database module backed by AsyncStorage.
4. **Implement Moment Capture Screen**
   - Add text input, voice toggle, photo option, tag selection, and save action.
5. **Implement Daily Summary Screen**
   - Add summary text, key moments list, actionable insights, and primary actions.
6. **Run Checks**
   - Attempt dependency install/type checks and report environmental blockers.
